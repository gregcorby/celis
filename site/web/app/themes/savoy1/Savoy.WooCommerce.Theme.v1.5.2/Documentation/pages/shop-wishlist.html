Shop → Wishlist

Savoy comes with a custom product Wishlist plugin. Clicking a "heart" icon in the product listing will toggle the product's visibility in the wishlist.

Follow the guide below to setup the Wishlist.

Wishlist Setup

If the "Savoy Theme - Wishlist" plugin is not installed, follow the Plugin Installation guide first.

Navigate to Pages → Add New in the WordPress admin.

  1. Enter a page title.
  2. Paste the following shortcode into the Text editor: [nm_wishlist]
  3. Finally, click the Publish button.
That's it, products added to the wishlist will now be listed on the page you created.
