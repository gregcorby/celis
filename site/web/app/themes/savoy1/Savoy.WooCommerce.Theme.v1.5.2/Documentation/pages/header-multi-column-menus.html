Header → Multi Column Menus

Savoy has support for multi-column menus. Follow the directions below to start building one.

You can manage and create the multi-column menus by navigating to Appearance → Menus.

1. Enable "CSS classes" field

First we need to make sure the "CSS Classes" field is displaying:

  1. Click the Screen Options tab in the top-right corner of the WordPress admin.
  2. Check the CSS Classes box.
2. Enter Menu Classes

Select the Main or Right menu in the WordPress menu editor. Toggle-open one of the top-level menu item's, and enter "megamenu col-3" in the "CSS Classes" field. This will display a 3 column menu.

To change the number of columns in the menu, simply replace "col-3" with "col-(1, 2 or 4)".

3. Organize Menu Items

You can now start organizing your multi-column menu items.
