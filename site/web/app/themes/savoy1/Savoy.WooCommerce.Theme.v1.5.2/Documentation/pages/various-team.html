Various → Team Members

Savoy comes with a custom Team Members plugin. The plugin adds a custom section to add members, and a Team page builder element.

1. Add/Edit Team Members

If the "Savoy Theme - Team Members" plugin is not installed, follow the Plugin Installation guide first.

To add/edit a team member, navigate to Team in the WordPress admin.

  1. Enter team member's name.
  2. Set a profile picture.
  3. Enter team member's status and social profile URL's.
2. Add to Page

Team members can be displayed on any page on your site:

  1. Navigate to Pages in the WordPress admin menu and select a page (or add a new one).
  2. Using the Page Builder, click any + button and select the Team element.
  3. Finally, specify the element settings in the modal window and save.
