Shop → Color Filter Widget

Savoy comes with a custom widget for filtering products by color. Follow the guide below to get started.

1. Add Color Attribute

First we need to add the "color" attribute that will be used by the widget.

Navigate to Products → Attributes in the WordPress admin.

  1. Enter "Color" into the Name field in the "Add New Attribute" form.
  2. Click the Add Attribute button.
2. Add Color Attribute Terms

After the "color" attribute has been created we need to add some terms (colors in this case).

  1. Click the Configure terms button for the "color" attribute entry.
  1.  
  2. Enter a color-name into the Name field in the "Add New Color" form.
  3. Click the Add New Color button.

Repeat this step for every color you want to add.

3. Add Color Widget

The color widget can now be added to the filters widget panel in the Shop.

Navigate to Appearance → Widgets in the WordPress admin.

  1. Drag and drop the "WooCommerce Color Filter" widget into the "Shop" container.
  2. Set a color for each term/color added using its color-picker.
  3. Click the Save button.
Done, you can now add filterable colors to your products.

The Color attributes have been set-up, and can now be added to your products.

If you're not familiar with how to do this, you can follow the official WooCommerce instructions to get started (see "Adding attributes for variations").
