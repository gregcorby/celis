Blog → Image Galleries

Savoy has support for displaying custom image galleries in your Blog archive/listing and posts.

Blog Gallery

Display images from the first post gallery instead of the "Featured Image" in your Blog archive/listing.

To configure the Blog Gallery settings, navigate to Theme Settings → Blog in the WordPress admin.

Custom WordPress Gallery

Display a custom image gallery slider instead of the default WordPress gallery in your blog posts.

To configure the Custom WordPress Gallery settings, navigate to Theme Settings → Blog in the WordPress admin.
