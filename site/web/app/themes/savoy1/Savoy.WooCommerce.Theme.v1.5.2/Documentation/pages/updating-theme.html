Updating → Auto Update

In this guide you will learn how to update the theme using the Envato Toolkit plugin. This plugin lets you easily update from the WordPress admin, without manually upload any files.

Step 1: Navigate to Envato Toolkit from your WordPress admin.

Step 2:

Note: This step requires an Envato Marketplace API key. Login to your ThemeForest account, and follow this guide to get one.

  1. Enter your Envato Marketplace username into the Marketplace Username field.
  2. Enter your Envato Marketplace API key into the Secret API Key field.
  3. Click the Save Settings button.

Final Step:

You should now be connected to your ThemeForest account, and ready to update the theme:

  1. Click the Themes tab on top of the page.
  2. And finally, click the update automatically link in the notification box to update to the latest version.

That's it!
